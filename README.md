# 🐞 Console-Based Bug Tracking System

A **menu-driven, console-based application** designed to simulate real-world bug tracking workflows. This system provides role-based functionality for **Testers, Developers, and Managers**, ensuring efficient bug reporting, assignment, and resolution. Built with simplicity and clarity in mind, it demonstrates how collaborative teams manage the lifecycle of software bugs.

---

## 🚀 Features

### 🔹 Tester Role
- Report new bugs with details (title, description, priority, status).
- View all reported bugs.
- Add comments to existing bugs for clarification or updates.

### 🔹 Developer Role
- Claim bugs assigned or available in the system.
- Update bug status (*In Progress*, *Resolved*, etc.).
- View bugs assigned specifically to them.
- Resolve bugs once fixes are implemented.

### 🔹 Manager Role
- Access a dashboard overview of all bugs.
- Assign bugs to developers based on workload or expertise.
- Generate detailed bug reports.
- Search and filter bugs by **status, priority, or assignee**.

## 🎯 Purpose
This project provides a **hands-on demonstration of bug lifecycle management** in a collaborative environment. By simulating real-world roles, it helps developers, testers, and managers understand how issues are tracked and resolved in professional software projects. Its console-based design ensures simplicity while reinforcing core programming concepts such as **role-based access, menu-driven navigation, and CRUD operations**.

---

## 📊 Benefits
- Encourages structured bug reporting and resolution.
- Reinforces teamwork and accountability through role separation.
- Enhanced reporting with export options (CSV, PDF).
- Provides managers with actionable insights via reports and search filters.
- Serves as a practical project for students and developers learning **Python, software engineering practices, and system design**.

## ⚠️ Disclaimer
This project is intended for **educational and demonstration purposes only**. It is not a production-ready bug tracking system but a prototype showcasing role-based workflows and console-driven interaction.
